| â, ê, î, ô, û Â, Ê, Î, Ô, Û        | Ctrl+Shift+^ (circunflexo), a letra |

| ä, ë, ï, ö, ü, ÿ, Ä, Ë, Ï, Ö, Ü, Ÿ | Ctrl+Shift+: (dois-pontos), a letra |

| æ, Æ                               | Ctrl+Shift+&, a ou A                |
| œ, Œ                               | Ctrl+Shift+&, o ou O                |
| ç, Ç                               | Ctrl+, (vírgula), c ou C            |
| ð, Ð                               | Ctrl+' (apóstrofo), d ou D          |
| ø, Ø                               | Ctrl+/, o ou O                      |
| ¿                                  | Alt+Ctrl+Shift+?                    |
| ¡                                  | ALT+CTRL+SHIFT+!                    |
| ß                                  | CTRL+SHIFT+&, s                     |
| '                                  | ALT + '                             |
